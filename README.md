# Angular CRUD

### Dependent 
- Ionic 5
- Angular 12
- Chart JS
- API Layer
- Capacitor

### Installation instruction

1. https://github.com/simplifyjs/ionic-angular-chartjs.git
2. In your terminal, run `npm install`
3. Run `ionic serve`
